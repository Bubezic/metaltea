Установка и запуск

Чтобы запустить проект локально, выполните следующие шаги:

1.  Скачайте архив и разархивируйте его в любое удобное место.
  
2.  Запустите Visual Studio Code

3.  Запустите проект ThirdAgeUniversity

4.  Откройте Terminal и перейдите в путь PS C:\Users\Name\Desktop\ThirdAgeUniversity\vk-mini\mini-app> 

5.  Введите команду npm install
                               
6.  Откройте десктопную версию сайта ВКонтакте и войдите, используя свою учётную запись.
    Перейдите на портал разработчиков https://dev.vk.com и выберите "Приложения" в верхнем меню.

7.  В поле Тип выберите Мини-приложение.
    Укажите название приложения. Вы можете указать любое имя и изменить его позже в разделе Настройки → Информация.
    Выберите категорию мини-приложения.
    Нажмите Создать.

    Во всплывающем окне Подтверждение действия выберите способ подтверждения и одобрите действие.

8.  Запустите Пуск → Все приложения → Node.js → Install Additional Tools for Node.js и следуйте инструкциям на экране.

9.  В том же Terminal введите команды npm install @vkontakte/vk-tunnel --include=dev
                                      npm run start -- --host

10. Откройте новый Terminal и введите команду npm run tunnel

11. В командной строке вы увидите ссылку для авторизации https://user344667137-smdlcam2.tunnel.vk-apps.com/, перейдите и авторизуйтесь, 
    на вашем экране должно появиться сообщение "Приложение VK Tunnel получило доступ к вашему аккаунту".

12. Вернитесь в окно командной строки и подтвердите действие: введите Y или y, или нажмите Enter.
    Если всё в порядке и VK Tunnel запущен, вы получите сообщение со ссылкой на мини-приложение:
    
    https://user743784479-cx6qjdfo.wormhole.vk-apps.com/
    Это значит, что VK Tunnel отобразил в интернете веб-сервер, работающий на вашем локальном компьютере. После этого можно запустить мини-приложение внутри ВКонтакте.

13. В браузере перейдите в настройки мини-приложения. 
    Откройте раздел Настройки → Размещение. В блоке Десктопная версии сайта включите Режим разработки и укажите полученную на предыдущем шаге ссылку как URL для разработки. 
    То же самое можно сделать и для других платформ.

14. Запустить Мини-приложение в верхнем правом углу. 
    Примечание, кнопка запуска будет названа так же, как ваше приложение.
